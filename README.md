# Mike's Blockchain Explorer

This app allows you to query the blockchain, and receive transaction data, either starting at a latest block, or querying a range of blocks.

To run the app:

Run `npm install`

Run `npm start`

Go to `localhost:8000` in your browser

To test the app:

Run `npm test`
